"Aqui puedes codificar y decodificar mensajes ocultos

Esta página es ideal para el aprendizaje para niños donde servirá para codificar mensajes que vaya a utilizar en cualquier juego que requiera ocultar o cifrar mensajes. 1º deberá introducir el mensaje que quiere codificar. El mensaje solo deberá contener letras, espacios, comas y puntos. 2º Elija el tipo de cifrado que quiere utilizar. 3º Pulse el botón de codificar. Una vez realizados estos pasos se mostrará el mensaje codificado y se incluirá el código que permitirá a los jugadores descifrar dicho mensaje.
